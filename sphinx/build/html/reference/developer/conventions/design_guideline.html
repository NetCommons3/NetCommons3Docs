1.5. デザインガイドライン¶

目次

  • bootstrapを利用
  • できるだけシンプル

利用者に優しいNetCommonsにするために、ガイドラインに従ってデザインをお願いします。

1.5.1. bootstrapを利用¶

bootstrapのテーマを切り替えるだけで簡単に見た目が変えれるように、原則bootstrapが提供しているCSS、画像アイコン、JavaScriptを利用しましょう。

1.5.2. できるだけシンプル¶

限られた狭いところで何でもやる必要はありません。利用者のエラーを誘発しないように、専用の画面に遷移する等をして、シンプルな画面に仕上げましょう。

1.5.3. 何をすればよいか分かる¶

便利で使いやすく利用者が次に何をするべきなのか分かるようにしましょう。

1.5.4. 視線の移動は直線的に、近く左から右、上から下に¶

出来るだけ自然な動作を促すことで、項目の見落としや誤解によるヒューマンエラーを防ぎましょう。
